# Build
Pastikan Java JDK 8 (java SE-1.8) terinstall

Buka Project structure pada intelliJ `ctrl + shift + alt + s` 

Pada bagian Project pastikan Project SDK menggunakan java 1.8 dengan SDK Default 8-Lambdas
